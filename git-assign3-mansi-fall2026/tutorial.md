# Git Tutorial Summary

I completed an interactive Git tutorial to learn the basics of version control.

## What I Learned

- Git is a version control system used to track changes in code.
- A repository stores project files and their history.
- Commits are used to save changes with meaningful messages.
- Branching allows working on features without affecting the main code.
- GitHub is used to host repositories online and collaborate with others.

## Commands I Learned

- git init: Initialize a repository
- git add: Add files to staging area
- git commit: Save changes
- git push: Upload changes to GitHub
- git pull: Fetch updates from repository

## Conclusion

This tutorial helped me understand how Git works and how developers collaborate using GitHub.
